# nottia

Public newsletter site for Nottia Weekly Briefing.

This repository contains only the public GitHub Pages site for nottia.

## Structure

- `index.html` - newsletter home page.
- `issues/` - published weekly issue pages.
- `assets/` - optional public images, styles, and scripts.
- `.nojekyll` - disables Jekyll processing for GitHub Pages.

Private working notes, prompts, drafts, and source references live outside this
repository in the parent nottia project directory.
